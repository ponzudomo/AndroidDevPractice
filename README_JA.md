# Android Developers — 練習リポジトリ

English version: [README.md](README.md)

## 概要

このリポジトリは、公式の **[Android Developers トレーニングコース](https://developer.android.com/courses)** を受講しながら作成した練習用プロジェクトをまとめたものです。

実際にコードを書くことで学習内容を定着させることを主な目的としています。トップレベルのフォルダはコースのプロジェクトに対応しています。

---

## リポジトリ構成

```
Practice/
├── docs/                          # プロジェクト全体のドキュメント
│   ├── GIT_WORKFLOW.md            # ブランチ戦略・コミットメッセージ規則
│   └── ...                        # ロードマップなど今後追加予定
└── AndroidBasicsCompose/          # コース: Android Basics with Compose
    ├── Unit1/                     # Unit 1 — はじめての Android アプリ
    │   ├── HappyBirthday/
    │   └── README.md
    ├── Unit2/                     # Unit 2 — アプリ UI の構築
    │   ├── DiceRoller/
    │   └── README.md
    └── ...                        # コースの進行に合わせて追加予定
```

> フォルダの階層はコースの構成に対応しています：**コース → ユニット → プロジェクト**。
> 各ユニットフォルダには、そのユニットに含まれるプロジェクトをまとめた `README.md` を置いています。
