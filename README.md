# Android Developers — Practice

日本語版: [README_JA.md](README_JA.md)

## Overview

This repository contains practice projects created while following the official **[Android Developers training courses](https://developer.android.com/courses)**.

The primary goal is to **support and reinforce learning** through hands-on coding. Each top-level folder corresponds to a single course project.

---

## Repository Structure

```
Practice/
├── docs/                          # Project-wide documentation
│   ├── GIT_WORKFLOW.md            # Branching strategy & commit message guide
│   └── ...                        # More docs will be added (e.g. roadmap)
└── AndroidBasicsCompose/          # Course: Android Basics with Compose
    ├── Unit1/                     # Unit 1 — Your First Android App
    │   ├── HappyBirthday/
    │   └── README.md
    ├── Unit2/                     # Unit 2 — Building App UI
    │   ├── DiceRoller/
    │   └── README.md
    └── ...                        # More units will be added as the course progresses
```

> The folder hierarchy follows the structure of the courses: **Course → Unit → Project**.
> A `README.md` is placed in each Unit folder to summarize the projects it contains.
